# server.java

## Description

A connections server with an HTTP module

## Features

##### HTTP
* Serves files in a directory

## Dependencies

##### Execution
* Java 1.7

##### Testing
* JUnit
* Hamcrest

##### Build tool
* Gradle

## Run tests
`gradle test --info`

## Compile
`gradle jar` will compile the *.jar file at `build/libs/server-0.1.0.jar`.
